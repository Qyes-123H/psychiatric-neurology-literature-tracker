## 2. 研究对象与疾病背景
- **主要疾病**：颞叶癫痫（Temporal Lobe Epilepsy, TLE）
- **共病精神症状**：抑郁症（Depression）
- **研究分组**：TLE合并抑郁组（TLE-D）、单纯TLE组（TLE）、健康对照组（HC）
- **评估工具**：17项汉密尔顿抑郁量表（HAMD-17）

## 3. 主要发现
1.  **GFAP（胶质纤维酸性蛋白）**：TLE-D组血浆GFAP水平显著高于TLE组和HC组，且TLE组高于HC组。在控制年龄等变量后，TLE-D组仍显著高于TLE组。GFAP升高提示星形胶质细胞损伤或激活。
2.  **NfL（神经丝轻链）**：TLE-D组和TLE组的NfL水平均显著高于HC组，但TLE-D组与TLE组之间无显著差异。NfL升高提示轴突损伤，与癫痫本身相关，而非抑郁的特异性标志物。
3.  **p-Tau181（磷酸化tau蛋白181）**：TLE-D组血浆p-Tau181水平显著高于TLE组和HC组，而TLE组与HC组无差异。p-Tau181升高提示神经原纤维缠结病理，与抑郁共病相关。

## 1. 一句话结论
嗅觉识别障碍是社区老年人群中，与阿尔茨海默病（AD）和路易体病（LBD）早期神经退行性改变均相关的非特异性标志物，即使在认知正常的人群中也存在。

## 2. 研究对象与疾病背景
- **研究对象**：787名社区居住的、年龄≥65岁、无痴呆或帕金森病的老年人。
- **疾病背景**：阿尔茨海默病（AD）和路易体病（LBD）。研究关注的是这两种疾病在临床前或极早期阶段的共同和差异性表现，特别是嗅觉功能、认知、运动体征（帕金森样症状）和血浆生物标志物之间的关系。

## 3. 主要发现
1.  **嗅觉与认知的广泛关联**：嗅觉识别能力（BSIT评分）越差，在所有认知领域（记忆、执行功能、语言、视空间、处理速度）的得分都越低，呈现“非分化模式”（即不特异于某个认知域）。
2.  **嗅觉与帕金森样体征的关联**：较差的嗅觉识别能力与步速减慢、静止性震颤以及快速眼动期睡眠行为障碍（RBD）样症状显著相关。
3.  **APOE4基因的调节作用**：在APOE4基因携带者中，嗅觉与记忆功能的关联性更强，提示嗅觉障碍可能更早地指向AD病理通路。
4.  **在认知正常者中仍显著**：即使在认知完全正常（CDR=0）的亚组中，上述嗅觉与认知、运动体征的关联大部分仍然存在。
5.  **与血浆生物标志物的关联**：探索性分析发现，较差的嗅觉评分与血浆p-tau217（AD特异性标志物）和神经丝轻链（NfL，神经轴突损伤标志物）水平升高相关。

3.  **杏仁核萎缩与视幻觉相关**：探索性分析发现，杏仁核某些亚区的体积缩小可能与DLB患者视幻觉症状的出现有关，但该发现尚需进一步验证。
4.  **与AD的对比**：研究证实了AD患者中已知的海马和杏仁核萎缩模式，而DLB的萎缩模式在分布和程度上与AD有所不同，为鉴别诊断提供了新线索。

4.  **临床特征关联**：右侧杏仁核的KFA仅在SCD组中与年龄呈负相关，提示SCD患者的杏仁核对衰老更敏感。左侧杏仁核的FA仅在对照组中与焦虑症状呈负相关，提示在疾病早期，这种正常关联可能已经中断。

## 四、适合科室分享的方向

1. 老年或精神症状首发患者，需要持续关注痴呆、谵妄、自身免疫性脑炎、癫痫、代谢性脑病等器质性病因。
2. 痴呆患者的幻觉、妄想、激越、淡漠、抑郁和睡眠障碍，是精神科专科医院神经内科的重要交叉管理方向。
3. 抗精神病药、抗抑郁药、锂盐、丙戊酸等药物相关运动障碍、认知影响和癫痫风险，值得形成固定会诊思路。
